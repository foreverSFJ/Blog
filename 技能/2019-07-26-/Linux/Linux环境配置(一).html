前提条件：虚拟机已安装好CentOs7，安装指导：

1. 网卡配置

  • 使用用户名/密码登录：root/root

  • 查看本地ip地址
    ip addr

  • 结果显示

    说明本地网卡没有启用，则启动网卡：

    • 命令：
      vi /etc/sysconfig/network-scripts/ifcfg-ens33

    • 结果：

      说明网卡没有启用，使用vim编辑，把no改为yes即可

    • 重启网卡服务

      sudo service network restart

  • 此时再次查看，即可发现网卡已经启用

2. 远程连接

由于Linux使用时，各种路径转换极为不便，一般情况下都是用远程连接进行操作。

一般使用SecureCRT或者xshell进行远程控制，使用Filezilla或者xftp进行文件传输

我使用的是SecureCRT和Filezilla进行远程操作

2.1. SecureCRT连接

  • 文件/连接

  • 新建会话

  • 基本一路下一步，遇到有填的就填一下

  • 填主机名和用户名

  • 下一步

  • 点击完成，然后连接刚行创建的会话

  • 输入密码，完成连接
