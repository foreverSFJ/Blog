4.1 概述

常量：是指在Java程序中固定不变的量。

4.2 分类

类型 含义 数据举例
整数常量 直接写上的数字，没有小数点 0，1， 567， -9
浮点数常量 直接写上的数字，有小数点 0.0， -0.1， 2.55
字符常量 单引号引起来，只能写一个字符，必须有内容 ‘a’ ， ‘ ‘， ‘好’
字符串常量 双引号引起来，可以写多个字符，也可以不写 “A” ，”Hello” ，”你好” ，””
布尔常量 只有两个值 true ， false
空常量 只有一个值 null

4.3 练习

需求：输出各种类型的常量。
